Install Java 11 Early-Access Build

Download an appropriate OpenJDK Early-Access build for your operating system.

Once installed, you can use the java command from your command line.

Check if the correct java version is installed:


$ java -version
openjdk version "11-ea" 2018-09-25
OpenJDK Runtime Environment 18.9 (build 11-ea+22)
OpenJDK 64-Bit Server VM 18.9 (build 11-ea+22, mixed mode)

You need to set the JAVA_HOME environment variable to a JDK 11 installation directory. You can follow this guide to set JAVA_HOME for your platform.

Note: If you have newer or older versions of Java installed along with JDK 11, you need to make sure to set the JAVA_HOME environment variable points to JDK 11.
