Run HelloWorld using JavaFX 11

Download an appropriate OpenJFX Early-Access Build for your operating system (11-ea+23 is the latest build at this moment).

You can now compile and run JavaFX 11 applications from the command line. In order to do so, you have to point the module-path to the location of the JavaFX Modules, e.g. 


export PATH_TO_FX=/path/to/javafx-sdk-11/lib

Compiling the code (e.g. use HelloFX.java) goes as follows: 


javac --module-path $PATH_TO_FX --add-modules=javafx.controls HelloFX.java

Make sure to add the required modules, keeping into account transitive dependencies are solved (hence there is no need to add the javafx.graphics module, since it is required by the javafx.controls module).

Next, run the application using: 


java --module-path $PATH_TO_FX --add-modules=javafx.controls HelloFX
